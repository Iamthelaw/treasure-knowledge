* Debugging
** Quickly add queries logging in Django
#+BEGIN_SRC python
  LOGGING = {
      'version': 1,
      'disable_existing_loggers': False,
      'handlers': {
          'console': {
              'level': 'DEBUG',
              'class': 'logging.StreamHandler',
          }
      },
      'loggers': {
          'django.db.backends': {
              'handlers': ['console'],
              'level': 'DEBUG',
          },
      }
  }
#+END_SRC
